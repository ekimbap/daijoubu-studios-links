# Daijoubu Studios Links

Static Linktree-style landing page for Daijoubu Studios, built from the LittleLink static HTML/CSS structure.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.

## GitHub Pages

This site is designed to publish from the `main` branch root using GitHub Pages.
